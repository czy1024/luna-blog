    # 左边参数名称为SPU，右边具体参数为SKU

    • 属性都是与三级分类组织，每一个分类一个属性列表，列表包含许多组，每一个组包含具体某一个参数，例如，手机->iphonx->64G && 银色->{主芯片，存储}
